# pandas.core.window.expanding.Expanding.count

Expanding.count(numeric_only=False)[source](https://github.com/pandas-dev/pandas/blob/main/pandas/core/window/expanding.py#L182-L205)#(https://pandas.pydata.org/pandas-docs/version/2.3.3/reference/api/pandas.core.window.expanding.Expanding.count.html#pandas.core.window.expanding.Expanding.count)
    Calculate the expanding count of non NaN observations.

Returns:
    Series or DataFrame
    Return type is the same as the original object with np.float64 dtype.

Examples

```
>>> ser = pd.Series([1, 2, 3, 4], index=['a', 'b', 'c', 'd'])
>>> ser.expanding().count()
```
