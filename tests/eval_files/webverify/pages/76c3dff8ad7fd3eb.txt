# pandas.RangeIndex.start

    The value of the start parameter (0 if this was not supplied).

Examples

```
>>> idx = pd.RangeIndex(5)
>>> idx.start
0

```

```
>>> idx = pd.RangeIndex(2, -10, -3)
>>> idx.start
2

```
