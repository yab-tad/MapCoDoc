# pandas.core.window.rolling.Rolling.rank

Rolling.rank(method='average', ascending=True, pct=False, numeric_only=False)[source](https://github.com/pandas-dev/pandas/blob/main/pandas/core/window/rolling.py#L2604-L2677)#(https://pandas.pydata.org/pandas-docs/version/2.3.3/reference/api/pandas.core.window.rolling.Rolling.rank.html#pandas.core.window.rolling.Rolling.rank)
    Calculate the rolling rank.

Added in version 1.4.0.

Parameters:
    method : {'average', 'min', 'max'}, default 'average'
    How to rank the group of records that have the same value (i.e. ties):

- average: average rank of the group
- min: lowest rank in the group
- max: highest rank in the group

ascending : bool, default True
    Whether or not the elements should be ranked in ascending order.

pct : bool, default False
    Whether or not to display the returned rankings in percentile
form.

numeric_only : bool, default False
    Include only float, int, boolean columns.

Added in version 1.5.0.

Returns:
    Series or DataFrame
    Return type is the same as the original object with np.float64 dtype.

Examples

```
>>> s = pd.Series([1, 4, 2, 3, 5, 3])
>>> s.rolling(3).rank()

3    2.0

5    1.5
dtype: float64

```

```
>>> s.rolling(3).rank(method="max")
```
