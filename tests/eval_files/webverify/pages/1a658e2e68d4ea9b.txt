# torch.autograd.Function.jvp

static Function.jvp(ctx, *grad_inputs)[source](https://github.com/pytorch/pytorch/blob/v2.9.1/torch/autograd/function.py#L446)#(https://docs.pytorch.org/docs/2.9/generated/torch.autograd.Function.jvp.html#torch.autograd.Function.jvp)
    Define a formula for differentiating the operation with forward mode automatic differentiation.

This function is to be overridden by all subclasses.
It must accept a context ctx as the first argument, followed by
as many inputs as the forward()(https://docs.pytorch.org/docs/2.9/generated/torch.autograd.Function.forward.html#torch.autograd.Function.forward) got (None will be passed in
for non tensor inputs of the forward function),
and it should return as many tensors as there were outputs to
forward()(https://docs.pytorch.org/docs/2.9/generated/torch.autograd.Function.forward.html#torch.autograd.Function.forward). Each argument is the gradient w.r.t the given input,
and each returned value should be the gradient w.r.t. the
corresponding output. If an output is not a Tensor or the function is not
differentiable with respect to that output, you can just pass None as a
gradient for that input.

You can use the ctx object to pass any value from the forward to this
functions.

Return type
    Any(https://docs.python.org/3/library/typing.html#typing.Any)
