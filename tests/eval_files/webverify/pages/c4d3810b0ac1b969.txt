# pandas.io.formats.style.Styler.set_caption

Styler.set_caption(caption)[source](https://github.com/pandas-dev/pandas/blob/main/pandas/io/formats/style.py#L2313-L2347)#(https://pandas.pydata.org/pandas-docs/version/2.3.3/reference/api/pandas.io.formats.style.Styler.set_caption.html#pandas.io.formats.style.Styler.set_caption)
    Set the text added to a <caption> HTML element.

Parameters:
    caption : str, tuple, list
    For HTML output either the string input is used or the first element of the
tuple. For LaTeX the string input provides a caption and the additional
tuple input allows for full captions and short captions, in that order.

Returns:
    Styler

Examples

```
>>> df = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
>>> df.style.set_caption("test")  

```
