Styler.highlight_max(subset=None, color='yellow', axis=0, props=None)[source](https://github.com/pandas-dev/pandas/blob/main/pandas/io/formats/style.py#L3228-L3282)#(https://pandas.pydata.org/pandas-docs/version/2.3.3/reference/api/pandas.io.formats.style.Styler.highlight_max.html#pandas.io.formats.style.Styler.highlight_max)
    Highlight the maximum with a style.

Parameters:
    subset : label, array-like, IndexSlice, optional
    A valid 2d input to DataFrame.loc[<subset>], or, in the case of a 1d input
or single key, to DataFrame.loc[:, <subset>] where the columns are
prioritised, to limit data to before applying the function.

color : str, default 'yellow'
    Background color to use for highlighting.

axis : {0 or 'index', 1 or 'columns', None}, default 0
    Apply to each column (axis=0 or 'index'), to each row
(axis=1 or 'columns'), or to the entire DataFrame at once
with axis=None.

props : str, default None
    CSS properties to use for highlighting. If props is given, color
is not used.

Added in version 1.3.0.

Returns:
    Styler
